FrontEnders ™ Studio

É um aplicativo de desktop para projetar e prototipar sites.

Ele vem com um grande número de componentes internos, que você pode arrastar e soltar para montar páginas da Web responsivas. O aplicativo é construído sobre a estrutura extremamente popular do Bootstrap e exporta HTML limpo e semântico.

Uma ferramenta completa para o seu dia a dia.

Para algumas coisas, arrastar e soltar não é suficiente. É por isso que o Bootstrap Studio oferece controle total sobre sua marcação quando você precisar. Você pode importar e editar CSS, SASS, JavaScript e HTML em nosso editor Sublime Text-like.

Crie designs responsivos sem muito esforço.

O aplicativo possui ferramentas especializadas para trabalhar com a grade do Bootstrap. Crie, redimensione e desloque colunas com facilidade e aplique classes de visibilidade responsivas
